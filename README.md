Converts Priyom calendar to mem.txt, user.txt and playlist.txt for ATS Mini V4 (1-4?) with HJ Berndt FW.

Start script. Upload mem.txt, user.txt and playlist.txt to ATS Mini and then: Good luck.

All transmitters will be hearable for 5 minutes. After 5 Minutes Radio is muting. That's a function of playlist. Duration can be changed here: f.write(f"{time_str} {idx} 5\n") I'm too lazy to se up a variable for it.
