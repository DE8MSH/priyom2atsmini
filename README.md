Converts Priyom calendar to mem.txt, user.txt and playlist.txt for ATS Mini V4 (1-4?) with HJ Berndt FW.

http://www.hjberndt.de/dvb/pocketSI4735DualCoreDecoder.html

Start script. Upload mem.txt, user.txt and playlist.txt to ATS Mini and then: Good luck.

All transmitters will be hearable for 5 minutes. After 5 Minutes Radio is muting. That's a parameter of playlist.txt. Duration can be changed in code (DURATION).
